# LLM Agent VAPT Framework

## Overview
AI-native VAPT framework for testing LLM applications, AI agents, and orchestration systems.

## Key Features
- OWASP Top 10 for LLMs aligned testing
- Prompt injection and jailbreak detection
- RAG poisoning and retrieval manipulation testing
- Agent/tool abuse simulations
- Multi-step adversarial attack chains
- Risk-based reporting
